Este projeto apresenta uma análise completa de uma campanha de marketing direto de um banco português. O objetivo é transformar dados brutos em inteligência acionável . Através da análise exploratória em Python e da construção de um dashboard interativo no Power BI , identifiquei o perfil do cliente com maior probabilidade de adesão a um depósito a prazo. A solução final é uma ferramenta de BI que permite à equipe de marketing tomar decisões estratégicas baseadas em dados.




Análise Exploratória em Python

Análise dos dados utilizando Pandas para manipulação, e Matplotlib/Seaborn para visualização. Esta etapa foi crucial para descobrir os padrões e os principais fatores de influência na conversão, como sazonalidade e histórico do cliente.

Dashboard Interativo em Power BI

Tradução dos insights da análise em um dashboard dinâmico e de fácil utilização. A ferramenta permite a segmentação de clientes em tempo real, a identificação de tendências e a exploração dos dados através de filtros e visuais interativos.

Resumo do Projeto de Análise

Objetivo

Identificar o perfil do cliente ideal e os fatores-chave (sazonalidade, histórico, etc.) que impactam a taxa de adesão a um depósito a prazo.

Tecnologias

Python (Pandas, Matplotlib) para análise exploratória e Microsoft Power BI para a construção do dashboard interativo e visualização de dados.

Dados

Utilizado o dataset "Bank Marketing" da UCI, contendo mais de 45 mil registros de uma campanha de telemarketing de um banco português.

Tratamento dos Dados

Limpeza de dados, tratamento de categorias "unknown" e criação de colunas personalizadas no Power BI (ex: 'Status da Dívida') para enriquecer a análise.

Visualização de Dados

Criação de um painel claro e objetivo no Power BI, com KPIs e visuais que destacam os insights mais importantes para a tomada de decisão.

Resultado

Um dashboard que permite à equipe de marketing segmentar melhor os clientes, otimizar o timing das campanhas e maximizar o retorno sobre o investimento.

Análise Exploratória e Principais Insights (Python)

Antes da construção do dashboard, foi realizada uma Análise Exploratória de Dados (EDA) em Python, no Jupyter Notebooks, para investigar as relações entre as variáveis e identificar os principais fatores de influência na adesão à campanha. Abaixo estão os principais achados.


O Processo de Análise Exploratória de Dados (EDA)

O processo seguiu as seguintes etapas:

  • Carregamento e Limpeza dos Dados: O dataset foi carregado em um DataFrame Pandas. A primeira etapa envolveu a verificação de valores ausentes, dados duplicados e a conversão de tipos de dados para garantir a consistência.
  • Análise Univariada: Cada variável foi analisada individualmente para entender sua distribuição. Gráficos de barras foram usados para variáveis categóricas (como `job`, `marital`) e histogramas para variáveis numéricas (como `age`, `balance`).
  • Análise Bivariada: O núcleo da análise, onde cada variável preditora foi cruzada com a variável alvo (`y`) para calcular a taxa de adesão de cada categoria. Isso foi feito para todas as variáveis relevantes, como escolaridade, endividamento e mês do contato.
  • Análise de Segmentos Combinados: Na etapa final, os insights das análises anteriores foram combinados para criar e analisar um "super segmento" de clientes com alto potencial de conversão, validando as hipóteses levantadas.

Principais Fatores de Influência Descobertos

A análise revelou que certos fatores tinham um impacto desproporcionalmente alto na chance de um cliente aderir à campanha. Os mais significativos foram:

  • Histórico do Cliente (`poutcome`): O fator preditivo mais forte. Clientes que já tiveram sucesso em campanhas passadas tiveram uma taxa de conversão de 64.7% , mostrando que a fidelidade é crucial.
  • Sazonalidade (`month`): O timing da campanha é fundamental. Os meses de Março, Setembro, Outubro e Dezembro apresentaram picos de adesão, com taxas que chegaram a superar 50% , em contraste com o pior mês, Maio (apenas 6.7%).
  • Perfil de Endividamento (`loan`, `housing`): Clientes sem empréstimos pessoais ou imobiliários converteram significativamente mais. A ausência de um financiamento imobiliário, por exemplo, mais do que dobrava a chance de adesão (16.7% vs 7.7%).
  • Profissão (`job`): Certos grupos profissionais se mostraram muito mais receptivos. Estudantes (28.7%) e Aposentados (22.8%) lideraram com as maiores taxas de conversão.

Conclusão: O Perfil do Cliente Ideal

A conclusão central da análise exploratória é que uma abordagem de marketing massificada é ineficiente. A combinação dos insights permitiu delinear um perfil do cliente com maior probabilidade de adesão:

O cliente ideal é um estudante ou aposentado , com ensino superior , sem dívidas de empréstimos e, idealmente, com um histórico de sucesso em campanhas passadas. A abordagem a este cliente deve ser feita preferencialmente via celular durante os meses de pico da campanha.

Esta definição do perfil ideal foi a principal diretriz para a construção do dashboard no Power BI, permitindo que a equipe de marketing segmente e priorize seus esforços de forma estratégica para maximizar os resultados.
